# ART GALLERY News

A responsive news website for an art gallery, built with HTML and CSS.

## About the project
This project is a news site for an art gallery featuring a hero section 
with the latest exhibition, a sidebar with recent articles, and a 
"Recently Added" grid with 9 news cards.

## What I built
- Responsive layout using Flexbox and CSS Grid
- Hero section with a featured article and sidebar
- News cards grid (3 columns)
- Animated ticker with latest headlines
- Hover effects on cards (lift + image zoom)
- Fade-in animation on page load

## Tools & techniques
- HTML5 & CSS3
- CSS Grid and Flexbox
- CSS animations and transitions
- Google Fonts (Lato)
- Git & GitHub
